Random Access: Camera Roll

A performance generated by chance, memory, and control.

Concept

This performance uses my camera roll as a dataset and Photo Roulette as a random generator. Each image appears without my control, forcing me to respond in real time. A coin flip determines whether I speak or remain silent, creating tension between exposure and privacy.

System

Input: Random photo

Chance: Coin flip

Output: Speech or silence

Script




INPUT: random photo (Photo Roulette)
CHANCE: coin flip (heads / tails)
TIME: 2 minutes






Walk in silently
AirPlay phone to screen
Set 2 minute timer
Remain silent






WHILE timer is running:

    Show photo

    Flip coin

    IF time < halfway:

        IF heads:
            explain photo

        IF tails:
            stay silent

    IF time >= halfway:

        IF heads:
            stay silent

        IF tails:
            say one word about photo

    WAIT 5–10 seconds






Stop when timer ends
No explanation

Reflection

This performance showed how little control I have over what memories appear. Some moments are easy to share, while others feel uncomfortable. The coin flip creates a clear divide between speaking and silence.
